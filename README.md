# flix-cli-hello-world

## Description
A POC for flix programming language.

## Tech stack
- java

## Docker stack
- openjdk:18-slim
- rzuckerm/flix

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
